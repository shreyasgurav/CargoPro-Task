DEVELOPMENT WORKFLOW
====================

1. Run Tests:
   flutter test

2. Start App (Web):
   flutter run -d chrome

3. Start App (Android):
   flutter run -d android

4. Format Code:
   dart format .

5. Analyze:
   flutter analyze

6. Build Web:
   flutter build web --release

7. Build APK:
   flutter build apk --release
